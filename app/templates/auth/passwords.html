Password Details

When creating a staff member for the first time, the system automatically generates a password for them. If the new staff member has a birthday, their password will be:

{Birthday Year}-{First letter of their first name}{Last Name}-{Birthday month as number 1 - 12}
So a staff member named John Doe born on 05/25/1994 will have the following password:
1994-jdoe-5

It is possible to create a new staff member without a birthdate. If the staff member's birthday field is empty, their password will be:

generated-placeholder-pw-2096

If a user cannot get into their account, admin users are able to reset a user's password.
